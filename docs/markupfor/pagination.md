# Markup for Pagination component

A pagination navigates between a list of results.

```html
<nav aria-label="Pagination">
    <ol>
        <li><a href="#">Previous</a></li>
        <li><a href="#">1</a></li>
        <li><a href="#">2</a></li>
        <li><a href="#">3</a></li>
        <li><a href="#">Next</a></li>
    </ol>
</nav>
```
