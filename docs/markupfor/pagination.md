# Markup for Pagination component

Pagination is a navigation that indicates a series of related content exists in multiple pages.

```html
<nav aria-label="Pagination">
    <ol>
        <li><a href="#">Previous</a></li>
        <li><a href="#">1</a></li>
        <li><a href="#">2</a></li>
        <li><a href="#">3</a></li>
        <li><a href="#">Next</a></li>
    </ol>
</nav>
```
